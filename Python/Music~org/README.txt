
WINDOWS ONLY

INSTRUCTIONS:

1. Create a folder you wish to organise you music into (including the executable).
2. Add music files into this folder and or leave them in the downloads folder.
3. Execute the program and choose wether you want 'pedantic' mode on.

NOTE: 

Executable can be found in the 'dist' folder.
This program organises by genre; hence, illegally downloaded music maybe ignored.
